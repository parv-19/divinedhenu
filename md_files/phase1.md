Now create the complete visual design system for this incense/fragrance ecommerce website.

Use this color palette:
- Background: #F7F1E8
- Card background: #FFF8EF
- Primary text: #3B2A1A
- Secondary text: #6D5C4D
- Accent gold: #B88746
- Border: #E8D9C5

Typography:
- Use serif-style headings through Tailwind classes.
- Use clean readable body text.
- Headings should feel premium and calm.
- Buttons should be rounded-full with soft hover animation.

Update index.css and Tailwind config if needed.

Create reusable components:
1. Container
2. SectionHeading
3. PrimaryButton
4. SecondaryButton
5. Badge
6. ProductCard

ProductCard should include:
- Product image
- Category badge
- Product name
- Mood tags like Calm, Focus, Prayer, Fresh, Sleep
- Price
- Quick Add button
- Soft hover effect

Keep the design mobile responsive.
Do not overuse bright colors.
The final UI should look premium, clean, and peaceful.
